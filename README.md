# This application displays a list of sports and their event outcomes

## To start back-end:
```
cd sports-api
bundle install
rails db:migrate
rails db:seed
rails s -p=3001
```

## To start front-end:
```
cd sports-client
npm install
npm start
```

Then navigate to `http://localhost:3000/`


## To run tests:
### Back-End
```
cd back-end
rspec
```
### Front-End
```
cd front-end
npm test
```
