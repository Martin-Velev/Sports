#1 This application displays a list of sports and their event outcomes

#2 To start back-end:
```
cd sports-api
bundle install
rails db:migrate
rails db:seed
rails s -p=3001
```

#2 To start front-end:
```
cd sports-client
npm install
npm start
```

Then navigate to `http://localhost:3000/`


#2 To run tests:
#3 Back-End
```
cd back-end
rspec
```
#3 Front-End
```
cd front-end
npm test
```
